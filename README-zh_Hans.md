===== Minecraft: MCBBS Edition =====

我最近比较忙，所以此项目基本不会更新。（除非我非常闲）

---

这是干什么的？

这是一个针对 Minecraft: Java Edition (版本 1.15+) 的一个资源包。

只是用来搞笑而已，没什么实际用途。

---

这个玩意干了些什么？

它修改了部分 Minecraft 中 "物品" 名 和 "生物" 名，还有一些其他的。

当然，这些修改后的名字借鉴于部分 MCBBS (www.mcbbs.net) 的用户。
